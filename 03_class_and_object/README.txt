[제출 방법]
1) 디렉토리 및 파일 구조를 아래와 같이 작성하고 최상위 디렉토리에 학번_이름으로 명명하시오. (예:201812344_홍길동)
   (맨 상위 디렉토리) 학번_이름
     -> (바로 아래 디렉토리) Money1, Money2 Fraction, GasPump 디렉토리와 H1_10_answer.txt 파일
     -> 각 서브 디렉토리에 main.cpp와 해당 클래스의 cpp와 h 파일 하나씩. 총 3개의 파일을 포함시킴
         예)  Money1 디렉토리
                 Money.h, Money.cpp, main.cpp
         예)  GasPump 디렉토리
                 GasPump.h, GasPump.cpp, main.cpp

     -> H1 ~ H10은 텍스트 파일 H1_10_answer.txt에 작성하여 맨 상위 디렉토리에 포함시킨다.
